HTML

  • The head contains information about the webpage (metadata)
  • The body element represents the visible content shown to the user
  • The header element represents introductory content, typically a group of introductory or navigational aids.
  • The footer element is placed at the bottom of a webpage and typically contains info about the author, copyright data or related links

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • A declaration contains the CSS property and the value of the property (e.g. color: blue)

Git

  • git status: checks what branch we are currently on + docs that have not been staged/committed
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git pull origin main: checks if your using the most up-to-date main branch
  • git add -A: stage all the changes made
  • git commit -m [message]: commit staged changes
  • git push origin [branch name]: push to git hub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A JavaScript function is executed when "something" invokes it (calls it).
  • A JavaScript function is a block of code designed to perform a particular task.
  • Conditional statements are used to perform different actions based on different conditions.
