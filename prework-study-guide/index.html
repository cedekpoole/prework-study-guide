HTML

  • The head contains information about the webpage (metadata)
  • The body element represents the visible content shown to the user
  • The header element represents introductory content, typically a group of introductory or navigational aids.
  • The footer element is placed at the bottom of a webpage and typically contains info about the author, copyright data or related links

Git

  • git status: checks what branch we are currently on + docs that have not been staged/committed
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git pull origin main: checks if your using the most up-to-date main branch
  • git add -A: stage all the changes made
  • git commit -m [message]: commit staged changes
  • git push origin [branch name]: push to git hub
